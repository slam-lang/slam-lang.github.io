Mem is full of helper functions related to memory management

class heapentry SIZE: 17b

a helper for storing data in the heap

prop heapentry.Addr SIZE: 8b POS: 0b

the address of the entry

prop heapentry.Flags SIZE: 1b POS: 8b

some flags for the entry

prop heapentry.Size SIZE: 8b POS: 9b

the size of the entry

prop heapentry.Data SIZE: 0b POS: 17b

the actual entry data

proc heapentry.isentry START: 1 RET: 1

returns true if the address is a heapentry

class mem SIZE: 0b

a wrapper for memory related procs

proc mem.copy START: 3 RET: 0

copies x bytes from one location to another

proc mem.eq START: 3 RET: 1

returns true of the next x bytes are equal

proc mem.zero START: 2 RET: 0

sets the next x bytes to zero

class heap SIZE: 0b

a wrapper for the heap

proc heap.allocpage START: 0 RET: 0

DONT USE. allocates another page to the heap

proc heap.init START: 0 RET: 0

DONT USE. initializes the heap

proc heap.map START: 0 RET: 0

prints the heap to stdout

proc heap.alloc START: 1 RET: 1

allocates x bytes in the heap and returns a pointer

proc heap.zalloc START: 1 RET: 1

allocates x bytes in the heap sets them to zero and returns a pointer

proc heap.free START: 1 RET: 0

frees a pointer in the heap, quits on failure

proc heap.tryfree START: 1 RET: 0

frees a pointer in the heap does nothing on failure

proc heap.realloc START: 2 RET: 1

resizes a pointer in the heap
