Stdlib is a assortment of basic utilities to speed up general life in slam.

class bool SIZE: 1b

A boolean value

class cstr SIZE: 0b

A string compatible with c, meaning it is NULL terminated

proc cstr.len START: 1 RET: 1

Returns the length of the string

proc cstr.print START: 1 RET: 0

Prints the string to STDOUT

proc cstr.cr START: 0 RET: 0

Prints a new line to STDOUT

proc cstr.println START: 1 RET: 0

Prints the string followed by a new line to STDOUT

proc cstr.eq START: 2 RET: 1

Returns true if the 2 top strings are equal

proc cstr.copy START: 2 RET: 2

Copy a string from a pointer to another

proc cstr.last START: 1 RET: 1

Returns the last character of a string

proc cstr.clear START: 1 RET: 1

Fills a string with \0

proc cstr.isnum START: 1 RET: 1

Returns true if a string is a number

proc cstr.tonum START: 1 RET: 1

Parses a string to a int, quits on failure

proc cstr.split START: 3 RET: 2

Splits a string on a deliminator

class int SIZE: 8b

Stores a integer

prop int.data SIZE: 8b POS: 0b

The integers value

proc int.cstr START: 1 RET: 1

Converts a int to a cstr

proc int.cstrhex START: 1 RET: 1

Converts a int to a cstr with hexadecimal

proc int.print START: 1 RET: 0

Prints a int to STDOUT

proc int.printhex START: 1 RET: 0

Prints a int in base 16 to STDOUT
